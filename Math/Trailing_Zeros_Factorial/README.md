# Trailing Zeroes in Factorial

## Problem Statement
Given an integer n, return the number of trailing zeroes in n!.

---

## Approach
- Count factors of 5 recursively
- Each 5 contributes to a trailing zero

---

## Concepts Used
- Recursion
- Number Theory
- Factorization

---

## Complexity
- Time: O(log n)
- Space: O(log n)

---

## Example
Input: 5  
Output: 1
